P3199 [HNOI2009] 最小圈

题目描述

考虑带权有向图 $G=(V,E)$ 以及 $w:E\rightarrow \mathbb{R}$，每条边 $e=(i,j)$（$i\neq j$，$i, j\in V$）的权值定义为 $w_{i,j}$。设 $n=|V|$。

$c=(c_1,c_2,\cdots,c_k)$（$c_i\in V$）是 $G$ 中的一个圈当且仅当 $(c_i,c_{i+1})$（$1\le i<k$）和 $(c_k,c_1)$ 都在 $E$ 中。称 $k$ 为圈 $c$ 的长度，同时记 $c_{k+1}=c_1$，并定义圈 $c=(c_1,c_2,\cdots,c_k)$ 的平均值为

$$ \mu(c)= \frac 1 k \sum\limits_{i=1}^{k} w_{c_i,c_{i+1}} $$

即 $c$ 上所有边的权值的平均值。设 $\mu'(G)=\min_c\mu(c)$ 为 $G$ 中所有圈 $c$ 的平均值的最小值。

给定图 $G=(V,E)$ 以及 $w:E\rightarrow \mathbb{R}$，求出 $G$ 中所有圈 $c$ 的平均值的最小值 $\mu'(G)$。

输入格式

第一行两个正整数，分别为 $n$ 和 $m$，并用一个空格隔开。其中 $n=|V|$，$m=|E|$ 分别表示图中有 $n$ 个点 和 $m$ 条边。

接下来 $m$ 行，每行三个数 $i,j,w_{i,j}$，表示有一条边 $(i,j)$ 且该边的权值为 $w_{i,j}$，注意边权可以是实数。输入数据保证图 $G=(V,E)$ 连通，存在圈且有一个点能到达其他所有点。

输出格式

一个实数 $\mu'(G)$，要求精确到小数点后 $8$ 位。

输入输出样例 #1

输入 #1

4 5
1 2 5
2 3 5
3 1 5
2 4 3
4 1 3

输出 #1

3.66666667

输入输出样例 #2

输入 #2

2 2
1 2 -2.9
2 1 -3.1

输出 #2

-3.00000000

提示：本题存在 $O(nm)$ 的做法，但是 $O(nm\log n)$ 的做法也可以通过。

分析

我们要求圈内权重之和和边数之和的比值的最小值，很明显是一个分数规划最小化问题，使用二分法搜索合法分数值x。这里我们可以把每条边的权重看作w-x*c，这样只需要最后的路径权重和小于等于0即可，转化为最小路径问题，并且这个最小路径又小于等于0，说明我们其实要做的就是判断这个环是不是一个负权环或者0权环。因为不是负权环或者0权环的话，环的权重和是不可能小于等于0的。判断是否有负权环的方法，自然想到使用SPFA算法。
代码如下

#include <bits/stdc++.h>
using namespace std;
const int N=3010;
const double eps=1e-10;
struct edge{int v;double w;};
vector<edge> e[N];
int n,m;
int cnt[N];
bool in_que[N];
double d[N];
bool SPFA(double x)
{
    for(int s=1;s<=n;s++)//有向图，图连通并不代表可以从任意一个节点访问其余所有节点，所以需要进行遍历找到可以通往所有其他节点的出发点
    {
        memset(cnt,0,sizeof(cnt));
        memset(in_que,false,sizeof(in_que));
        fill(d,d+N,1e18);
        queue<int> q;
        q.push(s);
        d[s]=0;cnt[s]=0;in_que[s]=true;
        while(!q.empty())
        {
            int cur=q.front();
            q.pop(),in_que[cur]=false;
            for(auto it=e[cur].begin();it!=e[cur].end();++it)
            {
                int v=it->v;
                double w=it->w-x;//等价转换边的权重
                if(d[v]-(d[cur]+w)>-eps)//等价于d[v]>=d[cur]+w，既可以判断负权环，也可以判断0权环
                {
                    d[v]=d[cur]+w;
                    cnt[v]=cnt[cur]+1;
                    if(cnt[v]>=n)return true;
                    if(!in_que[v])q.push(v),in_que[v]=true;
                }
            }
        }
    }
    return false;
}
int main()
{
    cin>>n>>m;
    for(int i=1;i<=m;i++)//建立邻接表
    {
        int a,b;
        double c;
        cin>>a>>b>>c;
        e[a].push_back({b,c});
    }
    double l=-1e7-5,r=1e7+5;
    while(r-l>eps)
    {
        double mid=l+(r-l)/2;
        if(SPFA(mid))r=mid;//如果存在负权环或者0权环，分数值成立，寻找更下值
        else l=mid;
    }
    cout<<fixed<<setprecision(8)<<r;
    return 0;
}

最后我们发现会TLE，因为使用bfs形式的SPFA算法一个一个枚举出发点进行判断实在是太慢了。那么接下来我就会介绍两种优化方式。

方式一：所有节点一次性入队

想象一下，我们创建一个虚拟节点，这个节点与所有的节点都相连，并且路径权重都为0。我们直接对这个虚拟节点进行松弛操作，就不需要去遍历寻找那一个可以通往所有顶点的出发点了。但是代价就是我们也无法算出最短路，因为虚拟节点是不存在的，最后就算没有负权环，dist[]数组中的值也是没有意义的。这种方法适用于图中需要进行负权环的判断而不需要最短路长度的情况。
代码如下

#include <bits/stdc++.h>
using namespace std;
const int N=3010;
const double eps=1e-13;
int n,m;
struct edge{int v;double w;};
vector<edge> e[N];
double d[N];
int cnt[N];
bool in_que[N];
bool SPFA(double x)
{
    fill(cnt,cnt+N,0);//全部初始化为0而不是1，cnt记录的是从真实节点出发经过的边数，虚拟节点不会对cnt产生影响
    fill(in_que,in_que+N,false);
    fill(d,d+N,0);//距离全部初始化为0
    queue<int> q;
    for(int i=1;i<=n;i++)//全部入队
    {
        q.push(i);
        in_que[i]=true;
    }
    while(!q.empty())
    {
        int cur=q.front();
        q.pop();in_que[cur]=false;
        for(auto it=e[cur].begin();it!=e[cur].end();++it)
        {
            int v=it->v;
            double w=it->w-x;
            if(d[v]-(d[cur]+w)>-eps)
            {
                d[v]=d[cur]+w;
                cnt[v]=cnt[cur]+1;//v和u都是真实节点，相当于就是全部节点入队后虚拟节点就不存在了
                if(cnt[v]>=n)return true;//所以这里判断的还是最多有n-1条边，否则就存在负权环
                if(!in_que[v])q.push(v),in_que[v]=true;
            }
        }
    }
    return false;
}
int main()
{
    cin>>n>>m;
    for(int i=1;i<=m;i++)
    {
        int a,b;
        double c;
        cin>>a>>b>>c;
        e[a].push_back({b,c});
    }
    double l=-1e7-5,r=1e7+5;
    while(r-l>eps)
    {
        double mid=l+(r-l)/2;
        if(SPFA(mid))r=mid;
        else l=mid;
    }
    cout<<fixed<<setprecision(8)<<r;
    return 0;
}
