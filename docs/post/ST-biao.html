ST 表

算法简介

在处理静态数组的区间查询问题时，我们经常遇到“区间最值查（RMQ）”需求。例如，多次询问一个长度为 ( N ) 的数组在子区间 ([L, R]) 内的最大值或最小值。如果每次查询都遍历区间，时间复杂度为 ( O(N) )，在查询次数较多时效率低下。

ST 表 正是为解决静态 RMQ 问题而设计的一种数据结构。它能够以 ( O(N log N) ) 的时间进行预处理，然后以 ( O(1) ) 的时间回答每次区间最值查询。注意是静态数组才可以使用，如果涉及数据修改那么ST表就会失效。

核心原理：倍增思想

ST 表的核心是倍增思想与动态规划预处理。

状态定义

我们定义一个二维数组 st[i][j]，它表示：从下标 ( i ) 开始，长度为 ( 2^j ) 的区间内的最值（最大值或最小值，根据需求决定）。
即st[i][j]=max(a[i],a[i+1],......a[i+2^j-1])这里的max也可以换成其他操作符，显然min可以，甚至gcd也可以。
显然，对于任意 ( i )，st[i][0] = a[i]（长度为 (2^0 = 1) 的区间就是元素本身）。

状态转移

长度为 (2^j) 的区间可以均分成两个长度为 (2^{j-1}) 的子区间，这两个子区间分别是：

  • 左半部分：从 ( i ) 开始，长度 ( 2^{j-1} )，对应 st[i][j-1]
  • 右半部分：从 ( i + 2^{j-1} ) 开始，长度 ( 2^{j-1} )，对应 st[i + 2^{j-1}][j-1]

因此，转移方程为：st[i][j]=max(s[i][j-1],s[i+2^(j-1)][j-1])

这个过程可以通过双重循环实现：外层枚举区间长度指数 ( j )，内层枚举起点 ( i )。

查询操作

当我们需要查询区间 ([L, R]) 的最值时，我们找到最大的整数 ( k )，使得两段长为2^k的子区间能完整覆盖整个所求区间。

通常直接取：k=log2(R-L+1)这里只取k的整数部分。

那么，区间 ([L, R]) 可以被两个有重叠的、长度均为 ( 2^k ) 的区间完全覆盖：

  • 第一个区间：从 ( L ) 开始，长度 ( 2^k )，对应 st[L][k]
  • 第二个区间：从 ( R - 2^k + 1 ) 开始，长度 ( 2^k )，对应 st[R - 2^k + 1][k]

两个区间可以重叠，但是一定不要有空隙。

因此查询结果即为：query(L,R)=max(st[L][K],st[R-2^k+1][k])

代码实现

#include <bits/stdc++.h>
using namespace std;
int n;
int st[25][10];
int mylog2[25];
int query(int L,int R)//查询操作
{
    int k=mylog2[R-L+1];算出能覆盖[L,R]的区间长度
    return max(st[L][k],st[R-(1<<k)+1][k]);
}
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        cin>>st[i][0];//2^0=1，所以st[i][0]就是数本身
    }

    for(int j=1;(1<<j)<=n;j++)//从小到大倍增区间长度
    {
        int len=(1<<(j-1));//每个区间分成两个长度相等的子区间
        for(int i=1;i+(1<<j)-1<=n;i++)
        {
            st[i][j]=max(st[i][j-1],st[i+len][j-1]);//区间最大值就是两个子区间最大值的较大者
        }
    }

    mylog2[1]=0;//先算出log2的值来实现O(1)的查询
    for(int i=2;i<=n;i++)
    {
        mylog2[i]=mylog2[i/2]+1;//递推公式
    }

    int l,r;
    cin>>l>>r;
    cout<<query(l,r)<<endl;
    return 0;
}
