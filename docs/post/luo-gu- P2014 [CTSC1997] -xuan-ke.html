P2014 [CTSC1997] 选课

题目描述

在大学里每个学生，为了达到一定的学分，必须从很多课程里选择一些课程来学习，在课程里有些课程必须在某些课程之前学习，如高等数学总是在其它课程之前学习。现在有 $N$ 门功课，每门课有若干学分，分别记作 $s_1,s_2,\cdots,s_N$，每门课有一门或没有直接先修课（若课程 $a$ 是课程 $b$ 的先修课即只有学完了课程 $a$，才能学习课程 $b$）。一个学生要从这些课程里选择 $M$ 门课程学习，问他能获得的最大学分是多少？

题目保证课程安排无冲突。（即不会有 $a$ 是 $b$ 的先修课， $b$ 也是 $a$ 的先修课这类情况存在。）

输入格式

第一行有两个整数 $N$， $M$ 用空格隔开 $(0 \leq N \leq 300, 1 \leq M \leq 300)$。
接下来的 $N$ 行，第 $i+1$ 行包含两个整数 $k_i$ 和 $s_i$， $k_i$ 表示第 $i$ 门课的直接先修课，$s_i$ 表示第 $i$ 门课的学分。若 $k_i=0$ 表示没有直接先修课 $(0 \leq k_i \leq N, 1 \leq s_i \leq 20)$。

数据保证至少存在一个 $k_i=0$，即至少一门课无先修课。

输出格式

只有一行，选 $M$ 门课程的最大学分。

输入输出样例 #1

输入 #1

7 4
2 2
0 1
0 4
2 1
7 1
7 6
2 2

输出 #1

13

分析

因为物品之间存在多重依赖关系，所以我们采用树状数组来存储数据，显然是一道树上背包的例题。因为可能不止有一门课没有前导课，导致树不连通，所以我们虚拟一个0节点作为所有课程的前导课程，也就是整棵树的根。这个节点的价值是0，重量也是0。其他课程价值就是学分，重量都是1。一般这种树上dp，我们都需要先dfs到叶子节点然后向上传播解决问题，这道题也不例外。同时这道题应该算是一个分组背包，每一棵子树都代表的是一组物品，dp数组中存放的也是这一种物品在不同容量下的多种存放方案。

dp数组含义

dp[i][j]表示考虑第i件物品，背包容量为j的时候所能容纳的最大价值。因为这道题有依赖关系，所以最后的答案就是被依赖物品在容量下的最大价值。

初始化

如果有物品u，那么dp[u][w[u]]到dp[u][j]的值都应该初始化为v[u]。

递推公式

对于节点u，枚举容量j从w[u]到[m]，因为我们至少需要w[u]的空间来存放被依赖物品，然后枚举分组背包的组合情况，即枚举从0到j-w[u]容量下该组物品所组成的最大价值。

for(int j=m;j>=w[u];j--)
{
    for(int k=0;k<=j-w[u];k++)
    {
        dp[u][j]=max(dp[u][j],dp[u][j-k]+dp[x][k]);
    }
}

代码

#include <bits/stdc++.h>
using namespace std;
const int N=305;
int n,m;
int w[N];
int v[N];
int dp[N][N];
vector<int> e[N];
void dfs(int u,int fa)
{
    for(int i=w[u];i<=m;i++)dp[u][i]=v[u];//初始化
    for(auto&s:e[u])
    {
        if(s==fa)continue;
        dfs(s,u);/先找到叶子节点
        for(int j=m;j>=w[u];j--)//枚举容量
        {
            for(int k=0;k<=j-w[u];k++)//枚举组合
            {
                dp[u][j]=max(dp[u][j],dp[u][j-k]+dp[s][k]);//状态转移
            }
        }
    }
}
int main()
{
    cin>>n>>m;
    for(int i=1;i<=n;i++)//建树
    {
        int k;
        cin>>k>>v[i];
        e[i].push_back(k);
        e[k].push_back(i);
        w[i]=1;//重量设置为1，但是w[0]=v[0]=0
    }
    dfs(0,0);
    cout<<dp[0][m];//0为虚拟节点，所有课程的前置课程
    return 0;
}
