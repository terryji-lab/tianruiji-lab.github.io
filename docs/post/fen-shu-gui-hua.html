分数规划

用途：求 分数的最值。

问题：每种物品有两个权值 $a_i$ 和 $b_i$，

选出若干个物品使得 $\frac{\sum a_i}{\sum b_i}$ 最大/最小。

等价：给出 $a_i$ 和 $b_i$，求一组 $w_i \in {0,1}$，（0代表不选，1代表选）

最大化或最小化 $\frac{\sum w_i \cdot a_i}{\sum w_i \cdot b_i}$。

若干个物品：满足给定 约束条件 的物品。

二分法

最大化：二分一个答案 $x$，

$$\frac{\sum w_i \cdot a_i}{\sum w_i \cdot b_i} \geq x$$

$$\sum w_i \cdot a_i - x \sum w_i \cdot b_i \geq 0$$

$$\sum w_i \cdot (a_i - x b_i) \geq 0$$

即所有选择的 $$\ (a_i - x b_i) $$ 之和大于等于0则成立，否则不成立。

最小化：二分一个答案 $x$，

$$\frac{\sum w_i \cdot a_i}{\sum w_i \cdot b_i} \leq x$$

$$\sum w_i \cdot (a_i - x b_i) \leq 0$$

即所有选择的 $$\ (a_i - x b_i) $$ 之和小于等于0则成立，否则不成立。

解题思路

  1. 明确 $a_i, b_i$ 是什么，把 $a_i - x b_i$ 作为新权值求和。
  2. 明确 约束条件 是什么，选择合适的算法求和。

例题

有 $n$ 个物品，每个物品有两个权值 $a$ 和 $b$。

你可以放弃 $k$ 个物品，选 $n - k$ 个物品，使得
$$\frac{\sum a}{\sum b}$$ 最大。保证分子小于分母。

第一行输入 $n$ 和 $k$，第二行输入 $n$ 个 $a_i$，第三行输入 $n$ 个 $b_i$。

输出答案乘 $100$ 后四舍五入到整数的值。

分析

要保证分数值最大，那么只需要使用 $$\sum w_i \cdot (a_i - x b_i) \geq 0$$ 作为条件进行二分搜索。为了使 $$\sum w_i \cdot (a_i - x b_i) $$ 最大，我们利用贪心思维，算出每个物品的权重后进行排序，按照权重从大到小进行选择。代码如下

#include <bits/stdc++.h>
using namespace std;
const double eps=1e-8;
int n,k;
double a[1010];
double b[1010];
double c[1010];//存储权重
bool check(double x)
{
	double s=0;
	for(int i=1;i<=n;i++)
	{
		c[i]=a[i]-x*b[i];//计算权重
	}
	sort(&c[1],&c[1]+n);//排序
	for(int i=n;i>=k+1;i--)//从大到小选择n-k件物品
	{
		s+=c[i];
	}
	return s>-eps;//浮点数精度
}
int main()
{
	cin>>n>>k;
	for(int i=1;i<=n;i++)
	{
		cin>>a[i];
	}
	for(int i=1;i<=n;i++)
	{
		cin>>b[i];
	}
	double l=0,r=1;//保证分子小于分母，右边界为1
	while(r-l>eps)
	{
		double mid=l+(r-l)/2;
		if(check(mid))l=mid;//如果满足，就搜索更大值
		else r=mid;
	}
	cout<<fixed<<setprecision(0)<<l*100;//四舍五入输出浮点数整数部分
	return 0;
}
