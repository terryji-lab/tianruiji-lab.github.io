# P2024 [NOI2001] 食物链

## 题目描述

动物王国中有三类动物 $A,B,C$，这三类动物的食物链构成了有趣的环形。$A$ 吃 $B$，$B$ 吃 $C$，$C$ 吃 $A$。

现有 $N$ 个动物，以 $1 \sim N$ 编号。每个动物都是 $A,B,C$ 中的一种，但是我们并不知道它到底是哪一种。

有人用两种说法对这 $N$ 个动物所构成的食物链关系进行描述：

- 第一种说法是 `1 X Y`，表示 $X$ 和 $Y$ 是同类。
- 第二种说法是 `2 X Y`，表示 $X$ 吃 $Y$。

此人对 $N$ 个动物，用上述两种说法，一句接一句地说出 $K$ 句话，这 $K$ 句话有的是真的，有的是假的。当一句话满足下列三条之一时，这句话就是假话，否则就是真话。

- 当前的话与前面的某些真的话冲突，就是假话；
- 当前的话中 $X$ 或 $Y$ 比 $N$ 大，就是假话；
- 当前的话表示 $X$ 吃 $X$，就是假话。

你的任务是根据给定的 $N$ 和 $K$ 句话，输出假话的总数。

## 输入格式

第一行两个整数， $N,K$ ，表示有 $N$ 个动物， $K$ 句话。

第二行开始每行一句话。格式见题目描述与样例。

## 输出格式

一行，一个整数，表示假话的总数。

## 输入输出样例 #1

### 输入 #1

```
100 7
1 101 1
2 1 2
2 2 3
2 3 3
1 1 3
2 3 1
1 5 5

```

### 输出 #1

```
3

```

## 说明/提示

对于全部数据， $1\le N\le 5 \times 10^4$ ， $1\le K \le 10^5$ 。
## 分析
带权并查集模板题，感觉好难，脑袋都冒烟了。和普通的并查集相比，带权并查集加入集合的元素之间会有特定的关系，需要开一个权重数组来保存。有两个点``x``和``y``，则这道题我们构造的权重关系为:
- ``F=(d[x]-d[y]+3)%3``
- ``F``为``1``则``x``捕食``y``
- ``F``为``2``则``y``捕食``x``
- ``F``为``0``则``x``与``y``是同类

关系构造好了我们该考虑并查集的问题了。首先要考虑的是在压缩路径时如何保证权值的合法性？我们要严格保证``d[x]``是``x``与其真实父亲节点的权重值，所以要对``find``函数进行一些修改。
```cpp
int find(int u)
{
    if(fa[u]==u)return u;
    int t=find(fa[u]);//递归到根部
    d[u]+=d[fa[u]];//返回加权节点权重
    return fa[u]=t;//当前节点的父亲接到根上去
}
```
我们知道每个集合在合并之前都是一个**独立的合法的**系统，我们要考虑怎么合并让这两个集合合并后依旧合法。比如``x``和``y``是同类，我们把``x``所在的集合合并到``y``所在的集合上去，那么合并后``x``和``y``到根的权值应该是相等的，即``d[fa[x]]=d[y]-d[x]``。如果``x``可以捕食``y``，那么``d[fa[x]]=d[y]-d[x]+1``表明 两点间的权值差为``1``。
下一个问题是我们如何判断题目输入的条件是不是合法？**我们只需要先判断``x``和``y``是不是在同一个集合中，如果在，就判断两个点是否符合题目给出的条件，如果不在，就将两个点按照题目给的条件加入一个集合**。
