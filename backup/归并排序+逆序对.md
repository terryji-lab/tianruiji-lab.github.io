# 归并排序
和快速排序一样，归并排序也使用了分治的思想。不同的是，快速排序是先在大区间进行分类，再在小区间进行分类，直到没有更小的区间。而归并排序是先让小区间有序，然后再合并有序的小区间。一个是自上而下的，一个是自下而上的。归并排序的优点在于我们已经知道两个有序数列合并的时间复杂度为O(n)，可以大大降低时间复杂度，并且是一个稳定的排序。
## 代码实现
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=1e5;
int a[N],b[N],n;
void msort(int l,int r)
{
    if(l==r)return;//最小区间只有一个数字，已经是有序区间，直接返回
    int mid=l+r>>1;
    msort(l,mid);
    msort(mid+1,r);//递归到最小区间

    int i=l,j=mid+1,k=l;//l是左区间起点，mid是左区间终点，mid+1是右区间起点，r是右区间终点
    while(i<=mid && j<=r)
    {
        if(a[i]<=a[j])b[k++]=a[i++];
        else b[k++]=a[j++];//区间合并
    }
    while(i<=mid)b[k++]=a[i++];
    while(j<=r)b[k++]=a[j++];//总有一个区间的元素先被选完，所以清理剩余元素，得到有序区间b
    for(int i=l;i<=r;i++)a[i]=b[i];//将b复制给a，让待排序数组a的区间[l,r]有序
}
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)cin>>a[i];
    msort(1,n);
    for(int i=1;i<=n;i++)cout<<a[i]<<' ';
    return 0;
}
```

# P1908 逆序对

## 题目描述

最近，TOM 老猫查阅到一个人类称之为“逆序对”的东西，这东西是这样定义的：对于给定的一段正整数序列，逆序对就是序列中 $a_i>a_j$ 且 $i<j$ 的有序对。知道这概念后，他们就比赛谁先算出给定的一段正整数序列中逆序对的数目。注意序列中可能有重复数字。

**Update：数据已加强。**

## 输入格式

第一行，一个数 $n$，表示序列中有 $n$ 个数。

第二行 $n$ 个数，表示给定的序列。序列中每个数字不超过 $10^9$。

## 输出格式

输出序列中逆序对的数目。

## 输入输出样例 #1

### 输入 #1

```
6
5 4 2 6 3 1

```

### 输出 #1

```
11
```

### 分析
我们想一下有序区间合并的过程，如果选择左区间的元素那么就是正常的，但是如果选择右区间的元素就说明一定存在逆序对，我们只需要对归并排序的代码做一点小小的修改就可以解决这道题目了。代码如下：
```cpp
#include <bits/stdc++.h>
#define int long long//记得开long long
using namespace std;
const int N=5e5+5;
int a[N],b[N],n,ans;
void msort(int l,int r)
{
    if(l==r)return;
    int mid=l+r>>1;
    msort(l,mid);
    msort(mid+1,r);

    int i=l,j=mid+1,k=l;
    while(i<=mid && j<=r)
    {
        if(a[i]<=a[j])b[k++]=a[i++];
        else b[k++]=a[j++],ans+=(mid-i+1);//因为左区间是有序递增的，所以如果右区间当前数值小于左区间当前数值，就说明右区间当前数值与左区间所有剩余数组都组成逆序对
    }
    while(i<=mid)b[k++]=a[i++];
    while(j<=r)b[k++]=a[j++];
    for(int i=l;i<=r;i++)a[i]=b[i];
}
signed main()
{
    cin>>n;
    for(int i=1;i<=n;i++)cin>>a[i];
    msort(1,n);
    cout<<ans;
    return 0;
}
```
