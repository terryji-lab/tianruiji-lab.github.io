# 最长公共子序列
## DP数组含义
``dp[i][j]``表示以``i``为最后一个元素下标的数组与以``j``为最后一个元素下标的数组的最长公共子序列的长度。
## 初始化
``dp[0][i]``与``dp[i]][0]``都初始化为``0``，表示一个数组为空的时候，整体的公共子序列长度一定为``0``。
## 递推公式
我们遍历``dp[][]``数组的顺序是先固定第一个数组的``i``，然后遍历第二个数组的``j``，然后更新``i``，最后遍历整个二维数组。我们用``a[]``和``b[]``来表示两个数组，如果``a[i]==b[j]``即当前位置的元素相同，那么就在原有的最长公共子序列基础上加一``dp[i][j]=dp[i-1][j-1]+1``，否则如果``a[i]!=b[j]``，那么当前状态的公共子序列长度就是另外两个状态中较长的那一个``dp[i][j]=max(dp[i-1][j],dp[i][j-1])``。最后整个二维数组的最后一个元素就是我们要求的答案。
## 代码
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=3000;
int dp[N][N];
int a[N];
int b[N];
int n;
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)cin>>a[i];
    for(int i=1;i<=n;i++)cin>>b[i];
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=n;j++)
        {
            if(a[i]==b[j])dp[i][j]=dp[i-1][j-1]+1;
            else dp[i][j]=max(dp[i-1][j],dp[i][j-1]);
        }
    }
    cout<<dp[n][n];
    return 0;
}

```
## 补充
如果要记录最长公共子序列的内容，我们可以引入数组``p[][]``来记录每一个公共元素的前驱元素。
- 如果元素来自左上角即``dp[i][j]=dp[i-1][j-1]``，那么``p[i][j]=1``
- 如果元素来自左即``dp[i][j]=dp[i][j-1]``，那么``p[i][j]=2``
- 如果元素来自上即``dp[i][j]=dp[i-1][j]``，那么``p[i][j]=3``
- 如果``p[i][j]==0``，说明已经是第一个元素

之后我们顺着``p[][]``数组倒序输出就可以打印出最长公共子序列的元素，输出的时候我们只输出``p[i][j]==1``的位置，只有这些位置才是匹配的在序列之中的相同元素，代码如下：
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=3000;
int dp[N][N];
int a[N];
int b[N];
int p[N][N];
int n;
void Print(int x,int y)
{
    if(p[x][y]==0)return;
    else if(p[x][y]==1)
    {
        Print(x-1,y-1);
        cout<<a[x]<<' ';
    }
    else if(p[x][y]==2)
    {
        Print(x,y-1);
    }
    else if(p[x][y]==3)
    {
        Print(x-1,y);
    }
}
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)cin>>a[i];
    for(int i=1;i<=n;i++)cin>>b[i];
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=n;j++)
        {
            if(a[i]==b[j])dp[i][j]=dp[i-1][j-1]+1,p[i][j]=1;
            else if(dp[i][j-1]>dp[i-1][j])dp[i][j]=dp[i][j-1],p[i][j]=2;
            else dp[i][j]=dp[i-1][j],p[i][j]=3;
        }
    }
    cout<<dp[n][n]<<'\n';
    Print(n,n);
    return 0;
}

```
# 最长公共子串
子串和子序列的差别就在于是否连续，子序列可以是不连续的，但是子串一定是连续的，所以我们要对状态转移方程进行修改，如果``a[i]!=b[j]``时，直接``dp[i][j]=0``，最后整个``dp[][]``数组中的最大值就是最长公共子串的长度。
## 代码
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=3000;
int dp[N][N];
int a[N];
int b[N];
int n,ans;
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)cin>>a[i];
    for(int i=1;i<=n;i++)cin>>b[i];
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=n;j++)
        {
            if(a[i]==b[j])dp[i][j]=dp[i-1][j-1]+1;
            else dp[i][j]=0;
            ans=max(ans,dp[i][j]);
        }
    }
    cout<<ans;
    return 0;
}
```
