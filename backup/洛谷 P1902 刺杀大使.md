# P1902 刺杀大使

## 题目描述

某组织正在策划一起对某大使的刺杀行动。他们来到了使馆，准备完成此次刺杀，要进入使馆首先必须通过使馆前的防御迷阵。

迷阵由 $n\times m$ 个相同的小房间组成，每个房间与相邻四个房间之间有门可通行。在第 $n$ 行的 $m$ 个房间里有 $m$ 个机关，这些机关必须全部打开才可以进入大使馆。而第 $1$ 行的 $m$ 个房间有 $m$ 扇向外打开的门，是迷阵的入口。除了第 $1$ 行和第 $n$ 行的房间外，每个房间都被使馆的安保人员安装了激光杀伤装置，将会对进入房间的人造成一定的伤害。第 $i$ 行第 $j$ 列 造成的伤害值为 $p_{i,j}$（第 $1$ 行和第 $n$ 行的 $p$ 值全部为 $0$）。

现在某组织打算以最小伤害代价进入迷阵，打开全部机关，显然，他们可以选择任意多的人从任意的门进入，但必须到达第 $n$ 行的每个房间。一个士兵受到的伤害值为他到达某个机关的路径上所有房间的伤害值中的最大值，整个部队受到的伤害值为所有士兵的伤害值中的最大值。现在，这个恐怖组织掌握了迷阵的情况，他们需要提前知道怎么安排士兵的行进路线可以使得整个部队的伤害值最小。

## 输入格式

第一行有两个整数 $n,m$，表示迷阵的大小。

接下来 $n$ 行，每行 $m$ 个数，第 $i$ 行第 $j$ 列的数表示 $p_{i,j}$。

## 输出格式

输出一个数，表示最小伤害代价。

## 输入输出样例 #1

### 输入 #1

```
4 2
0 0 
3 5 
2 4 
0 0 

```

### 输出 #1

```
3
```

## 分析
这是一道**二分**加**搜索**的题目，使用DFS还是BFS都可以解决，只不过使用DFS需要一点剪枝技巧，否则会爆栈。使用二分查找P，然后使用搜索检查在P的值下是否可以**连通**，若可以，就查找更小的P，否则就增大P，找到**满足条件的最小值**（一般使用右指针来查找答案区间最小值,左指针查找答案区间最大值）
代码如下
```cpp
#include <bits/stdc++.h>
using namespace std;
int n,m;
int grid[1010][1010];
bool used[1010][1010];//记录每个位置是否到过
int dx[]={1,0,0,-1};
int dy[]={0,1,-1,0};//四个方向
bool dfs(int x,int y,int p)
{
	if(x==n)return true;//如果到达第n行结束，返回真
	used[x][y]=true;//标记当前位置
	for(int i=0;i<4;i++)
	{
		int nx=x+dx[i];
		int ny=y+dy[i];//四向搜索
		if(nx >=1 && nx<=n && ny>=1 && ny<=m && used[nx][ny]==false && grid[nx][ny]<=p)//条件判断，只有在边界内并且值小于p才能通过
		{
			if(dfs(nx,ny,p))return true;//减枝，如果能通过直接返回true停止递归
		}
	}
	return false;//不能通过返回false
}
int main()
{
	int lef=0,rig=0;
	cin>>n>>m;
	for(int i=1;i<=n;i++)
	{
		for(int j=1;j<=m;j++)
		{
			cin>>grid[i][j];
			rig=max(rig,grid[i][j]);
		}
	}
	rig+=1;
	while(lef+1<rig)
	{
		memset(used,false,sizeof(used));
		int mid=lef+(rig-lef)/2;
		if(dfs(1,1,mid))rig=mid;//如果能通过就寻找更小的最大值，右指针左移
		else lef=mid;//否则左指针右移
	}
	cout<<rig;//右区间为合法区间，右指针为合法区间的最小值
	return 0;
}
```
