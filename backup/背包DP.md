# 背包DP
背包问题就是在给定的容量内使物品的价值最大化。这里我会介绍**01背包**，**完全背包**，**多重背包**，**分组背包**四种经典的背包DP。
## 01背包
可以说是背包问题梦的起点了。**01**的含义就是每个物品只有一个，不选就是``0``，选了就是``1``。
### 朴素算法
### DP数组含义
朴素算法使用一个二维dp数组来实现，``dp[i][j]``表示考虑前``i``件物品在背包容量为``j``的时候，背包所能容纳的最大价值。
### 初始化
``dp[0][j]``和``dp[i][0]``都初始化为``0``，因为在考虑第``0``件物品和背包容量为``0``的时候，所容纳价值一定为``0``。（这里物品的下标从``1``开始）
### 递推公式
我们先固定物品，考虑容量，然后更新物品。当我们在考虑第``i``件物品的时候，如果物品重量小于背包容量，我们有两个选择，拿或者不拿。如果大于背包容量，就只有不拿了。在状态转移方程中的表示就是：
- 如果背包容量``j<w[i]``，不能放入，最大价值和不考虑这个物品是一样的，``dp[i][j]=dp[i-1][j]``
- 如果背包容量``j>=w[i]``，可以放入，选择放和不放当中较大的那一个值，``dp[i][j]=max(dp[i-1][j],dp[i-1][j-w[i]]+v[i])``，这里``j-w[i]``就是放入物品后的背包剩余容量，我们用剩余容量的最大价值加上物品``i``的价值就得到了放入物品的最大价值。放入最大价值与不放入最大价值取最大值就是当前状态最大价值。
最后``dp[n][m]``就是所求的答案。
### 代码
```cpp
for(int i=1;i<=n;i++)
{
	for(int j=1;j<=m;j++)
	{
		if(j<w[i])dp[i][j]=dp[i-1][j];
		else dp[i][j]=max(dp[i-1][j],dp[i-1][j-w[i]]+v[i]);
	}
}
```
### 滚动数组优化
我们开一个二维数组，空间复杂度过大，可以只使用一个一维滚动数组来解决这个问题。
### DP数组含义
``dp[i]``表示容量为``i``时背包能容纳的最大价值。
### 初始化
``dp[]``刚开始全部初始化为``0``，表示考虑``0``件物品时，最大价值为``0``。
### 递推公式
如果放不下物品，那么``dp[j]=dp[j]``，不改变能容纳的最大值，否则``dp[j]=max(dp[j],dp[j-w[i]]+v[i])``，更新为选取与不选取当中较大的那一个值。这个这里一定要倒序遍历``dp[j]``，我们可以把数组划分为更新后和未更新的两个部分。更新前表示没有考虑这个物品，更新后表示考虑了这个物品。我们更新一个位置需要的是未更新的位置的状态，也就是还未考虑这个物品的状态，如果我们顺序遍历，那么就会反复使用更新后的位置，相当于一个物品考虑了多次，变成了完全背包问题，所以我们需要倒序遍历，始终使用未更新的部分去更新状态。
### 代码
```cpp
for(int i=1;i<=n;i++)
{
	for(int j=m;j>=w[i];j--)//这里我直接改变遍历范围，避免不必要的判断操作
	{
		dp[j]=max(dp[j],dp[j-w[i]]+v[i]);
	}
}
```
## 完全背包
完全背包与01背包非常类似，区别就在于完全背包中的物品可以被考虑无数次，所以只需要对代码进行一点点修改就好了，这里直接给出代码。
### 代码
```cpp
for(int i=1;i<=n;i++)
{
	for(int j=1;j<=m;j++)
	{
		if(j<w[i])dp[i][j]=dp[i-1][j];
		else dp[i][j]=max(dp[i-1][j],dp[i][j-w[i]]+v[i]);
	}
}
```
同样也可以使用一维滚动数组优化，只需要改变遍历顺序即可，代码如下：
```cpp
for(int i=1;i<=n;i++)
{
	for(int j=w[i];j<=m;j++)
	{
		dp[j]=max(dp[j],dp[j-w[i]]+v[i]);
	}
}
```

## 多重背包
多重背包与01背包的区别是01背包中每个物品只有一件，而多重背包中每个物品有多件。因此，01背包中选与不选的问题，到了多重背包中就成了选还是不选，选的话要选几件的问题了。实际上就是转化为01背包问题，对相同的每一件物品讨论取与不取的问题。
### 朴素算法
### 代码
```cpp
for(int i=1;i<=n;i++)
{
    for(int j=m;j>=w[i];j--)
    {
        for(int k=0;k<=s[i] && k*w[i]<=j;k++)
        {
            dp[j]=max(dp[j],dp[j-k*w[i]]+k*v[i]);
        }
    }
}
```
但是显然这样的时间复杂度很高。由于有一些物品是相同的，所以我们不用一件一件的来考虑，可以把它们合并起来，这就是我要介绍的二进制优化法。
### 二进制优化
我们把每种物品按照**2的n次方**来进行分组，依旧可以组合出任意数量的物品，考虑的时候也不需要一件一件进行考虑了，减少了物品的总数。
### 代码
```cpp
int len=0;
for(int i=1;i<=n;i++)
{
    cin>>vv>>ww>>s;
    for(int j=1;j<=s;j<<=1)//j从1开始，每次乘2
    {
        v[++len]=j*vv;
        w[len]=j*ww;
        s-=j;//减去拿走的物品数量
    }
    if(s)//如果有剩余，进行处理
    {
        v[++len]=s*vv;
        w[len]=s*ww;
    }
}
```
