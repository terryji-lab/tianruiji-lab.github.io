# 前缀和与差分

## 前缀和

前缀和一种预处理技术，把从1到i的元素之和存在``pre[i]``当中，能够在 **O(1)** 时间内求出任意一个**连续子数组**的元素之和。

对于一个长度为 $n$ 的数组 $a$，定义其**前缀和数组** $pre$ 为：

$$
pre[i] = \sum_{j=1}^{i} a[j] = a[1] + a[2] + \cdots + a[i]
$$

特殊地，我们通常令 $pre[0] = 0$。并且一般使用``1~based``下标。

### 一维前缀和公式

- 求前缀和
```cpp
pre[i]=pre[i-1]+a[i];
```
- 查询区间和``[L,R]``
```cpp
sum=pre[R]-pre[L-1];
```
### 二维前缀和
若要求出一个二维数组的**子数组**元素之和，可以使用二维前缀和
- 以``i`` ``j``为右下角元素的子数组元素和,使用容斥原理（也可以先对每一行做一次前缀和，再对每一列做一次前缀和）
```cpp
pre[i][j]=pre[i-1][j]+pre[i][j-1]-pre[i-1][j-1]+a[i][j];
```
- 查询子数组``[x1,y1]``为左上角``[x2,y2]``为右下角的子数组元素和
```cpp
sum=pre[x2][y2]-pre[x1-1][y2]-pre[x2][y1-1]+pre[x1-1][y1-1];
```
