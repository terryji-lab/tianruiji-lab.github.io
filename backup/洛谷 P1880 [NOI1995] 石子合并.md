# P1880 [NOI1995] 石子合并

## 题目描述

在一个圆形操场的四周摆放 $N$ 堆石子，现要将石子有次序地合并成一堆，规定每次只能选相邻的 $2$ 堆合并成新的一堆，并将新的一堆的石子数，记为该次合并的得分。

试设计出一个算法，计算出将 $N$ 堆石子合并成 $1$ 堆的最小得分和最大得分。

## 输入格式

数据的第 $1$ 行是正整数 $N$，表示有 $N$ 堆石子。

第 $2$ 行有 $N$ 个整数，第 $i$ 个整数 $a_i$ 表示第 $i$ 堆石子的个数。

## 输出格式

输出共 $2$ 行，第 $1$ 行为最小得分，第 $2$ 行为最大得分。

## 输入输出样例 #1

### 输入 #1

```
4
4 5 9 4
```

### 输出 #1

```
43
54
```

## 说明/提示

$1\leq N\leq 100$， $0\leq a_i\leq 20$ 。

## 分析
这道题和普通的区间dp的区别在于区间成了一个环，首位相连。为了解决这类问题，我们要引入**破环为链的方法，也就是复制整个区间，将两个完全一样的区间首位相连拼接起来，正常进行区间dp便可以覆盖成环的所有情况。** 其余还是区间dp三步走，区间长度，起点，分割点，可以解决这道题，代码如下：
```cpp
#include <bits/stdc++.h>
#define int long long
using namespace std;
const int N=205;
int dp_max[N][N];
int dp_min[N][N];
int pre[N];
int n,minn=1e18,maxn=-1e18;
signed main()
{
    memset(dp_max,-0x3f,sizeof(dp_max));//求最大全部初始化为无穷小
    memset(dp_min,0x3f,sizeof(dp_min));//求最小全部初始化为无穷大
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        cin>>pre[i];
        pre[n+i]=pre[i];//复制区间
    }
    for(int i=1;i<=2*n;i++)
    {
        pre[i]+=pre[i-1];//计算前缀和
        dp_max[i][i]=0;
        dp_min[i][i]=0;//和自己合并代价为0
    }
    for(int len=2;len<=n;len++)//枚举区间长度
    {
        for(int l=1;l+len-1<=2*n;l++)//枚举起点
        {
            int r=l+len-1;
            for(int i=l;i<r;i++)//枚举分割点
            {
                dp_max[l][r]=max(dp_max[l][r],dp_max[l][i]+dp_max[i+1][r]+pre[r]-pre[l-1]);
                dp_min[l][r]=min(dp_min[l][r],dp_min[l][i]+dp_min[i+1][r]+pre[r]-pre[l-1]);
            }
        }
    }
    for(int i=1;i<=n;i++)
    {
        minn=min(minn,dp_min[i][i+n-1]);
        maxn=max(maxn,dp_max[i][i+n-1]);//寻找长度为n的区间的最大最小值点
    }
    cout<<minn<<'\n'<<maxn;
    return 0;
}
```
