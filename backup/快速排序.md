# 快速排序
快速排序是基于**分治的思想**来实现的。
在数组中，我们选择一个基准值，然后将大于该基准值的数字放到一侧，小于基准值的数字放到一侧，把数组划分为了两段。然后我们继续对这两段子数组数组进行相同的操作，大于基准值放到一侧，小于基准值放到一侧，重复这个过程，直到每个子数组只有一个元素时，就可以得排序后的数组。一般我们选取的基准值就是当前子数组最中间的那一个元素。
记住这个过程是**不稳定的，也就是说相同的元素相对位置会发生变化**。
## 代码
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=5e5+5;
int a[N],n;
void quicksort(int l,int r)//l和r代表当前数组的左右区间
{
    if(l==r)return;//如果左右区间相等说明已经只有一个元素，直接返回
    int i=l-1,j=r+1,x=a[(l+r)/2];//x是选区的基准值
    while(i<j)//最后循坏停止时i与j重合，是大于x元素和小于x元素的分界点
    {
        do i++;while(a[i]<x);//找到大于等于x的元素
        do j--;while(a[j]>x);//找到小于等于x的元素
        if(i<j)swap(a[i],a[j]);//将大于等于x的元素放到右区间，小于等于x的元素放到左区间
    }
    quicksort(l,j);//对小于x的子数组进行排序
    quicksort(j+1,r);//对大于x的子数组进行排序
}
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)cin>>a[i];
    quicksort(1,n);
    for(int i=1;i<=n;i++)cout<<a[i]<<' ';
    return 0;
}
```
当然在cpp中我们一般直接使用``sort``函数来排序，如果需要稳定的排序就调用``stable_sort``。
