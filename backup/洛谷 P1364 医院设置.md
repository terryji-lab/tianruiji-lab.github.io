# P1364 医院设置

## 题目描述

设有一棵二叉树，如图：

<img width="869" height="396" alt="Image" src="https://github.com/user-attachments/assets/8b99688a-775c-4680-bd4c-2d0b1a05b457" />

其中，圈中的数字表示结点中居民的人口。圈边上数字表示结点编号，现在要求在某个结点上建立一个医院，使所有居民所走的路程之和为最小，同时约定，相邻节点之间的距离为 $1$。如上图中，若医院建在 $1$ 处，则距离和 $=4+12+2\times20+2\times40=136$；若医院建在 $3$ 处，则距离和 $=4\times2+13+20+40=81$。

## 输入格式

第一行一个整数 $n$，表示树的结点数。

接下来的 $n$ 行每行描述了一个结点的状况，包含三个整数 $w, u, v$，其中 $w$ 为居民人口数， $u$ 为左链接（为 $0$ 表示无链接）， $v$ 为右链接（为 $0$ 表示无链接）。

## 输出格式

一个整数，表示最小距离和。

## 输入输出样例 #1

### 输入 #1

```
5						
13 2 3
4 0 0
12 4 5
20 0 0
40 0 0

```

### 输出 #1

```
81
```

## 说明/提示

#### 数据规模与约定

对于 $100\%$ 的数据，保证 $1 \leq n \leq 100$， $0 \leq u, v \leq n$， $1 \leq w \leq 10^5$。

## 分析
这道题数据量不大，可以选择的算法很多，比如Floyd多源最短路，或者对每一个点进行Bfs枚举出最佳答案，但是这里我要介绍的是树形dp中的换根dp做法。首先我们介绍一下换根dp。
### 换根dp
换根dp，又称为二次扫描，一般需要两次dfs来实现。
第一次dfs相当于打表，属于前置工作，对数状数组进行预处理，获得诸如深度，点权和之类的信息。
第二次dfs就是枚举以每一个点为根的情形，进行状态转移，我们以深度和为例子，这里的``dp[u]``表示以``u``为根时的深度和，``sz[v]``表示以``v``为根的子树大小，``total``表示总的节点数量：
- dp[v]=dp[u]-sz[v]+total-sz[v]

解释一下，这里``v``是``u``的子节点，如果以``v``为根的话，在以``v``为根的子树上的所有节点深度减一，也就是减少``sz[v]``。同时，不在以``v``为根的子树上的节点深度加一，也就是增加``total-sz[v]``，就得到了这样的状态转移方程。
同时我们也很好理解，第一次dfs我们只需要随意选择一个点作为根，并且只算出这一个点的深度和就可以了，后续节点的深度和都根据这个节点转移而来。
具体可以看``P3478 [POI 2008] STA-Station``的题解。

现在我们回到这道题，我们只需要把上面例子中的深度和改为**人口x深度**和，然后状态转移的时候，更换根的时候子树上变为**人口x(深度-1)**，也就是**人口x深度-人口**，就是减去子树上的总人口数。同理，不在子树上就加上不在子树上的总人口数，就可以优雅的解决这一道题。代码如下：
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=105;
int n,w[N],sz[N],dp[N],total,ans=INT_MAX;
vector<int> e[N];
void dfs1(int u,int fa,int dep)
{
    dp[1]+=w[u]*dep;//距离人口乘积和，这里我们以1为根，只算一个点
    sz[u]=w[u];
    total+=w[u];//计算总人口数
    for(auto&v:e[u])
    {
        if(v==fa)continue;
        dfs1(v,u,dep+1);
        sz[u]+=sz[v];//计算子树人口和
    }
}
void dfs2(int u,int fa)
{
    for(auto&v:e[u])
    {
        if(v==fa)continue;
        dp[v]=dp[u]-sz[v]+total-sz[v];//状态转移
        dfs2(v,u);//枚举下一个节点
    }
}
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)//建树
    {
        int u,v;
        cin>>w[i]>>u>>v;
        if(u)e[u].push_back(i),e[i].push_back(u);
        if(v)e[v].push_back(i),e[i].push_back(v);
    }
    dfs1(1,0,0);//第一次dfs
    dfs2(1,0);//第二次dfs
    for(int i=1;i<=n;i++)ans=min(ans,dp[i]);//寻找最小值
    cout<<ans;
    return 0;
}
```
