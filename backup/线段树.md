# 线段树
线段树是一种常见的可以对区间进行动态查询和修改的数据结构。使用分而治之的思想实现，把整个数组划分为多个区间，每个区间再划分成更多小区间，最后形成一个树状结构。
## 准备工作
我们创建结构体作为树的节点，保存该节点的左右区间范围，区间数值总和还有lazy标记。
```cpp
struct node
{
    int l,r,sum,lazy;
}tr[4*N];
```
## pushup操作
pushup指我们对小的区间进行修改后的反向传播操作，可以对已修改的子区间的父区间进行更新。代码如下：
```cpp
void pushup(int p)
{
    tr[p].sum=tr[p*2].sum+tr[p*2+1].sum;
}
```
## pushdown操作
当我们对一个区间进行分割递归之前要先用pushdown操作把lazy标记正向传播下去，对区间进行修改。代码如下：
```cpp
void pushdown(int p)
{
    if(tr[p].lazy)
    {
        tr[p*2].sum+=tr[p].lazy*(tr[p*2].r-tr[p*2].l+1);
        tr[p*2+1].sum+=tr[p].lazy*(tr[p*2+1].r-tr[p*2+1].l+1);
        tr[p*2].lazy+=tr[p].lazy;
        tr[p*2+1].lazy+=tr[p].lazy;
        tr[p].lazy=0;
    }
}
```
## build操作
建树操作，使用递归实现，代码如下：
```cpp
void build(int p,int l,int r)
{
    tr[p]={l,r,w[l]};
    if(l==r)return;
    int m=(l+r)/2;
    build(p*2,l,m);
    build(p*2+1,m+1,r);
    pushup(p);
}
```
