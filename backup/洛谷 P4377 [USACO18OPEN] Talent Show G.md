# P4377 [USACO18OPEN] Talent Show G

## 题目描述

Farmer John 要带着他的 $n$ 头奶牛，方便起见编号为 $1\ldots n$，到农业展览会上去，参加每年的达牛秀！他的第 $i$ 头奶牛重量为 $w_i$，才艺水平为 $t_i$，两者都是整数。

在到达时，Farmer John 就被今年达牛秀的新规则吓到了：

（一）参加比赛的一组奶牛必须总重量至少为 $W$（这是为了确保是强大的队伍在比赛，而不仅是强大的某头奶牛），并且。

（二）总才艺值与总重量的比值最大的一组获得胜利。

FJ 注意到他的所有奶牛的总重量不小于 $W$，所以他能够派出符合规则（一）的队伍。帮助他确定这样的队伍中能够达到的最佳的才艺与重量的比值。

## 输入格式

第一行是两个整数，分别表示牛的个数 $n$ 和总重量限制 $W$。

第 $2$ 到 $(n+1)$ 行，每行两个整数，第 $(i + 1)$ 行的整数表示第 $i$ 头奶牛的重量 $w_i$ 和才艺水平 $t_i$。

## 输出格式

请求出 Farmer 用一组总重量最少为 $W$ 的奶牛最大可能达到的总才艺值与总重量的比值。

如果你的答案是 $A$，输出 $1000A$ 向下取整的值，以使得输出是整数（当问题中的数不是一个整数的时候，向下取整操作在向下舍入到整数的时候去除所有小数部分）。

请注意当问题的答案恰好是整数 $x$ 时，你的程序可能会由于**浮点数精度误差**问题最后得到一个 $x-\epsilon$ 的答案，向下取整后变为 $x-1$ 导致答案错误。这种情况下你可以在输出答案前给答案加上一个极小的值 $x\gets x+10^{-k}$ 来避免该问题。

## 输入输出样例 #1

### 输入 #1

```
3 15
20 21
10 11
30 31
```

### 输出 #1

```
1066

```

## 说明/提示

#### 样例解释

在这个例子中，总体来看最佳的才艺与重量的比值应该是仅用一头才艺值为 $11$、重量为 $10$ 的奶牛，但是由于我们需要至少 $15$ 单位的重量，最优解最终为使用这头奶牛加上才艺值为 $21$、重量为 $20$ 的奶牛。这样的话才艺与重量的比值为 $\frac{11+21}{10+20}=\frac{32}{30} = 1.0666\dots$，乘以 $1000$ 向下取整之后得到 $1066$。

## 分析
分数规划的题目。这道题的约束条件**所有奶牛的总重量不小于** $W$ ，不是简单的约束选择多少个元素，所以**不能直接使用贪心排序选择元素**。让我们回顾一下分数规划的模板，这道题我们要求一个**分数的最大值**，那么我们需要保证： $$\sum w_i \cdot (t_i - x w_i) \geq 0$$  这道题的分子是t，分母是w，我们可以考虑使用**01背包**的思想。
我们可以把每一头牛看作重量为 $$\ w_i \$$ ，价值为 $$\ (t_i - x w_i) \$$ 的一件物品，我们要求的就是在**总容量为W的情况下能容纳物品的最大价值**。事实上道题还要复杂一点，因为题目要求的是**所有奶牛的总重量不小于** $W$ ，而不是总容量为W，我们需要把所有**重量大于W的情况看作重量为W来处理**。最后我们**判断在容量大于等于W时价值的总和是否大于等于0，大于等于0成立，否则不成立**，之后使用二分继续搜索。
这里dp数组的初始化真的很难理解，网上的很多解释都很牵强。为什么dp[0]要初始化为0，其余位置初始化为负无穷？**我的评价是自己在草稿纸上模拟两遍然后直接背下来就完事了**。
代码如下
```cpp
#include <bits/stdc++.h>
using namespace std;

const double eps = 1e-8;
const int MAXW = 1010;

int n, W;
int w[255];
int t[255];
double v[255];
double dp[MAXW];

bool check(double x)
{
    for (int i = 1; i <= n; ++i)
    {
        v[i] = t[i] - x * w[i];//计算价值
    }
    dp[0] = 0;//初始化dp数组，容量为0时价值为0
    for (int j = 1; j <= W; ++j)
    {
        dp[j] = -1e18;//背包容量小于W视作不可达，使用负无穷表示
    }
    for (int i = 1; i <= n; ++i)//背包dp一维滚动数组，先遍历物品
    {
        for (int j = W; j >= 0; --j)//再倒序遍历重量，防止物品被重复使用
        {
            int k = min(W, j + w[i]);//如果重量j加上物品i的重量w[i]大于等于W，则统一合并为状态W
            dp[k] = max(dp[k], dp[j] + v[i]);//拿物品之前的状态为j，拿之后为k，保留状态k的最大值
        }
    }
    return dp[W] > -eps;//如果容量大于等于W时价值大于0说明当前分数值x合法
}

int main()
{
    cin >> n >> W;
    for (int i = 1; i <= n; ++i)
    {
        cin >> w[i] >> t[i];
    }
    double l = 0, r = 1000;
    while (r - l > eps)
    {
        double mid = l + (r - l) / 2;
        if (check(mid))//合法寻找更大数
        {
            l = mid;
        }
        else
        {
            r = mid;
        }
    }
    cout << int(r * 1000);//类型转换，向下取整，并且取右指针防止向下取整产生的误差
    return 0;
}
```
