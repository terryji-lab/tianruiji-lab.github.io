# P3199 [HNOI2009] 最小圈

## 题目描述

考虑带权有向图 $G=(V,E)$ 以及 $w:E\rightarrow \mathbb{R}$，每条边 $e=(i,j)$（$i\neq j$，$i, j\in V$）的权值定义为 $w_{i,j}$。设 $n=|V|$。

$c=(c_1,c_2,\cdots,c_k)$（$c_i\in V$）是 $G$ 中的一个圈当且仅当 $(c_i,c_{i+1})$（$1\le i<k$）和 $(c_k,c_1)$ 都在 $E$ 中。称 $k$ 为圈 $c$ 的长度，同时记 $c_{k+1}=c_1$，并定义圈 $c=(c_1,c_2,\cdots,c_k)$ 的平均值为

$$
\mu(c)= \frac 1 k \sum\limits_{i=1}^{k} w_{c_i,c_{i+1}}
$$

即 $c$ 上所有边的权值的平均值。设 $\mu'(G)=\min_c\mu(c)$ 为 $G$ 中所有圈 $c$ 的平均值的最小值。

给定图 $G=(V,E)$ 以及 $w:E\rightarrow \mathbb{R}$，求出 $G$ 中所有圈 $c$ 的平均值的最小值 $\mu'(G)$。

## 输入格式

第一行两个正整数，分别为 $n$ 和 $m$，并用一个空格隔开。其中 $n=|V|$，$m=|E|$ 分别表示图中有 $n$ 个点 和 $m$ 条边。

接下来 $m$ 行，每行三个数 $i,j,w_{i,j}$，表示有一条边 $(i,j)$ 且该边的权值为 $w_{i,j}$，注意边权可以是实数。输入数据保证图 $G=(V,E)$ 连通，存在圈且有一个点能到达其他所有点。

## 输出格式

一个实数 $\mu'(G)$，要求精确到小数点后 $8$ 位。

## 输入输出样例 #1

### 输入 #1
```
4 5
1 2 5
2 3 5
3 1 5
2 4 3
4 1 3
```
### 输出 #1
```
3.66666667
```

## 分析
我们要求圈内权重之和和边数之和的比值的最小值，很明显是一个**分数规划最小化问题**，使用**二分法搜索合法分数值``x``**。这里我们可以把每条边的**权重看作``w-x*c``**，这样只需要最后的**路径权重和小于等于0即可，转化为最小路径问题**。我们其实要做的就是**判断这个环是不是一个负权环或者0权环**。判断是否有负权环的方法，自然想到使用**SPFA算法**。
代码如下
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=3010;
const double eps=1e-10;
struct edge{int v;double w;};
vector<edge> e[N];
int n,m;
int cnt[N];
bool in_que[N];
double d[N];
bool SPFA(double x)
{
    for(int s=1;s<=n;s++)//有向图，图连通并不代表可以从任意一个节点访问其余所有节点，所以需要进行遍历找到可以通往所有其他节点的出发点
    {
        memset(cnt,0,sizeof(cnt));
        memset(in_que,false,sizeof(in_que));
        fill(d,d+N,1e18);
        queue<int> q;
        q.push(s);
        d[s]=0;cnt[s]=0;in_que[s]=true;
        while(!q.empty())
        {
            int cur=q.front();
            q.pop(),in_que[cur]=false;
            for(auto it=e[cur].begin();it!=e[cur].end();++it)
            {
                int v=it->v;
                double w=it->w-x;//等价转换边的权重
                if(d[v]-(d[cur]+w)>-eps)//等价于d[v]>=d[cur]+w，既可以判断负权环，也可以判断0权环
                {
                    d[v]=d[cur]+w;
                    cnt[v]=cnt[cur]+1;
                    if(cnt[v]>=n)return true;
                    if(!in_que[v])q.push(v),in_que[v]=true;
                }
            }
        }
    }
    return false;
}
int main()
{
    cin>>n>>m;
    for(int i=1;i<=m;i++)//建立邻接表
    {
        int a,b;
        double c;
        cin>>a>>b>>c;
        e[a].push_back({b,c});
    }
    double l=-1e7-5,r=1e7+5;
    while(r-l>eps)
    {
        double mid=l+(r-l)/2;
        if(SPFA(mid))r=mid;//如果存在负权环或者0权环，分数值成立，寻找更下值
        else l=mid;
    }
    cout<<fixed<<setprecision(8)<<r;
    return 0;
}
```

最后我们发现会TLE，因为使用bfs形式的SPFA算法一个一个枚举出发点进行判断实在是太慢了。那么接下来我会介绍两种优化的方法都可以通过这道题。
## 方法一：创建虚拟起点
因为是**有向图，纵使图是连通的也无法通过一个点遍历整个图**，因此一般的SPFA算法就会遇到反复尝试起点导致超时的问题。对此，我们可以创建一个**虚拟起点，它与所有的节点都相连，并且路径权重都为0**，在一开始我们对这个虚拟点进行松弛操作，**直接把所有节点都加入队列**，避免一次一次的重复SPFA。**当然使用这种方法也会让``d[]``数组失去意义就算负权环不存在**
- 虚拟起点对所有节点的边权为0，并且是有向图，没有节点指向虚拟起点**不会人为制造新的负权环**
- 从虚拟起点出发，**可以一次性覆盖所有节点**
- 任何位置存在负环**都可以从虚拟起点到达**

代码实现如下
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=3010;
const double eps=1e-10;
int n,m;
struct edge{int v;double w;};
vector<edge> e[N];
double d[N];
int cnt[N];
bool in_que[N];
bool SPFA(double x)
{
    fill(cnt,cnt+N,0);//计数初始化为0
    fill(in_que,in_que+N,false);
    fill(d,d+N,0);//距离初始化为0
    queue<int> q;
    for(int i=1;i<=n;i++)//全部入队
    {
        q.push(i);
        in_que[i]=true;
    }
    while(!q.empty())
    {
        int cur=q.front();
        q.pop();in_que[cur]=false;
        for(auto it=e[cur].begin();it!=e[cur].end();++it)
        {
            int v=it->v;
            double w=it->w-x;
            if(d[v]-(d[cur]+w)>-eps)
            {
                d[v]=d[cur]+w;
                cnt[v]=cnt[cur]+1;
                if(cnt[v]>=n)return true;//上面我们把cnt初始化为0，表示除去和虚拟起点的边外到达这个点最小距离经过的边，所以还是不大于n-1。也可以把上面的cnt初始化为1，这里就是不大于n。
                if(!in_que[v])q.push(v),in_que[v]=true;
            }
        }
    }
    return false;
}
int main()
{
    cin>>n>>m;
    for(int i=1;i<=m;i++)
    {
        int a,b;
        double c;
        cin>>a>>b>>c;
        e[a].push_back({b,c});
    }
    double l=-1e7-5,r=1e7+5;
    while(r-l>eps)
    {
        double mid=l+(r-l)/2;
        if(SPFA(mid))r=mid;
        else l=mid;
    }
    cout<<fixed<<setprecision(8)<<r;
    return 0;
}
```
## 方法二：深度优先搜索
这是用于判断负权环更常见的算法。我们从一个点出发，然后一条路走到黑。**如果下一个点能被当前点松弛，就前往下一个点，如果这个点还在递归过程中，就说明产生了负权环**。为什么呢？
- 可以松弛才前往下一个点，说明**沿着该路径权值和是减小也就是单调递减的**
- **遇到递归过程中的节点说明形成了环**

**一条权值和单调递减的环，也就是可以无限递减的环，当然是一个负权环**。
代码实现如下
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=3010;
const double eps=1e-10;
struct edge{int v;double w;};
int n,m;
bool used[N];//记录当前点是否在递归过程中
double d[N];
vector<edge> e[N];
bool dfs(int u,double x)
{
    used[u]=true;//点u进入递归
    for(auto it=e[u].begin();it!=e[u].end();++it)
    {
        int v=it->v;
        double w=it->w-x;
        if(d[v]-(d[u]+w)>-eps)//如果下一个点能被当前点松弛
        {
            d[v]=d[u]+w;
            if(used[v])return true;//如果v在递归过程中说明负权环存在
            if(dfs(v,x))return true;//对v递归调用
        }
    }
    used[u]=false;//u点递归结束
    return false;//只是从u出发找不到负权环，而不是u不在负权环内，可以自己画图模拟领会一下
}
bool check(double x)
{
    memset(d,0,sizeof(d));//还是使用虚拟起点的方法，初始化为0。但其实这里初始化为多少都可以。
    memset(used,false,sizeof(used));
    for(int i=1;i<=n;i++)//以每个点为起点寻找负权环
    {
        if(dfs(i,x))return true;
    }
    return false;
}
int main()
{
    cin>>n>>m;
    for(int i=1;i<=m;i++)
    {
        int a,b;
        double c;
        cin>>a>>b>>c;
        e[a].push_back({b,c});
    }
    double l=-1e7-5,r=1e7+5;
    while(r-l>eps)
    {
        double mid=l+(r-l)/2;
        if(check(mid))r=mid;
        else l=mid;
    }
    cout<<fixed<<setprecision(8)<<r;
    return 0;
}
```
