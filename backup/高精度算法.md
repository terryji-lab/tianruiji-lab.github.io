# 高精度算法

高精度算法主要用于处理**超出一般数据类型存储大小**的数字计算，主要思路就是**用数组来模拟竖式运算的过程**。我们都知道int类型的范围大概是-2^31到2^31，大概也就几十亿，如果我要算一个100位的数字加减乘除怎么办呢，这就需要用到高精度算法了。

常见的高精度运算有四种，分别为：高精度加法，高精度减法，高精度乘法，高精度除法。

但是要注意，其中加减乘都是**高精度数与高精度数**的运算，但是除法是**高精度与低精度数**的运算，这一点到后面会体现出来。

高精度算法一般有统一的步骤：
1.**使用字符串读入**高精度数字（因为字符串可以读取任意长度的数字）
2.**字符串翻转倒序存入数组**（要模拟竖式，需要低位对齐）
3.**从低位到高位进行模拟运算**（除了除法以外，除法是从高位到低位）
4.**倒序输出**（很多时候需要处理前导0）

接下来我将要详细的讲解这四种运算

---

## 高精度加法

从低位到高位进行的操作为 **累加->进位->存余**。这和我们小学学的竖式加法是一模一样的，比如计算56+98，我们先算6+8=14，取4进1，然后算5+9+1=15，所以结果是154。

代码如下

```cpp
#include <bits/stdc++.h>
using namespace std;
int A[505];
int B[505];
int C[505];
int main()
{
	string a,b;
	cin>>a>>b;
	int la=a.size();
	int lb=b.size();
	for(int i=0;i<la;i++)
	{
		A[i]=a[la-i-1]-'0';//倒序输入
	}
	for(int i=0;i<lb;i++)
	{
		B[i]=b[lb-i-1]-'0';
	}
	int lc=max(la,lb);
	for(int i=0;i<lc;i++)
	{
		C[i]+=A[i]+B[i];//累加
		C[i+1]+=C[i]/10;//进位
		C[i]%=10;//取余
	}
	if(C[lc])lc++;//处理结果长度
	for(int i=lc-1;i>=0;i--)
	{
		cout<<C[i];//倒序输出
	}
	return 0;
}
```

我来解释一下核心代码的部分，首先是**倒序输入**，为什么要倒序呢，因为竖式运算从低位开始，我们的数组下标0对应的是数字的最低位，这样方便进位的处理。比如数字"123"存成数组就是A[0]=3,A[1]=2,A[2]=1。

然后是**核心的运算逻辑**，先把对应位相加，存入C[i]，这时候C[i]可能大于10，然后通过**C[i+1]+=C[i]/10**来实现进位，**C[i]%=10**来取余数。

最后为什么要**判断if(C[lc])呢**，因为最高位可能产生进位，比如99+1=100，最高位进位后lc会变成3，这时候结果数组C的0、1、2位分别是0、0、1，需要正确处理。

## 高精度减法

需要先判断进行运算的数字哪一个比较大，保证**大数减小数**，使得计算结果为正数。如果原计算结果为负数的话需要提前输出负号。

从低位到高位的操作为：**求差->借位->存差**。

这里要注意一点，减法和加法不一样，加法不管谁大谁小结果都是正的，但是减法**必须大数减小数**，否则结果就是负数了，所以我们需要提前判断，如果被减数比减数小，就要交换两个数的位置并且输出一个负号。

代码如下

```cpp
#include <bits/stdc++.h>
using namespace std;
int A[100100];
int B[100100];
int C[100100];
int main()
{
    string a,b;
    cin>>a>>b;
    int la=a.size(),lb=b.size();
    for(int i=0;i<la;i++)
    {
        A[i]=a[la-i-1]-'0';
    }
    for(int i=0;i<lb;i++)
    {
        B[i]=b[lb-i-1]-'0';
    }
    if(!cmp(A,la,B,lb))//若A<B就交换，提前输出负号
    {
        swap(A,B);
        swap(la,lb);
        cout<<'-';
    }
    for(int i=0;i<la;i++)
    {
        if(A[i]<B[i])//借位操作
        {
            A[i]+=10;
            A[i+1]--;
        }
        C[i]=A[i]-B[i];
    }
    int lc=la;
    while(lc && C[lc]==0)lc--;//处理前导0
    for(int i=lc;i>=0;i--)
    {
        cout<<C[i];//倒序输出
    }
    return 0;
}
```

**核心的借位操作**是这样的，如果当前位被减数比减数小，就需要向高位借1，当作10加到当前位上。比如计算100-1，先看个位0-1不够，向十位借1变成10-1=9，十位变成0-0=0，百位1-0=1，结果是99。

最后那个**while循环是用来处理前导0**的，比如计算100-100，如果不处理前导0会输出"001"这样的结果，我们从最高位开始找第一个非零的位置，去掉前面的0。

## 高精度乘法

从低位到高位：**累加乘积->进位->求余**。

乘法需要注意的是每一次相乘得到的结果应该存在结果数组的**哪一个位置**。这个和加减法不一样，乘法的位置关系比较特殊，**A数组的第i位乘以B数组的第j位，结果要存到C数组的i+j位**。

代码如下

```cpp
#include <bits/stdc++.h>
using namespace std;
int A[2010];
int B[2010];
int C[4010];//注意结果数组的大小
int main()
{
    string a,b;
    cin>>a>>b;
    int la=a.size();
    int lb=b.size();
    for(int i=0;i<la;i++)
    {
        A[i]=a[la-i-1]-'0';
    }
    for(int i=0;i<lb;i++)
    {
        B[i]=b[lb-i-1]-'0';
    }
    for(int i=0;i<la;i++)
    {
        for(int j=0;j<lb;j++)
        {
            C[i+j]+=A[i]*B[j];//累加乘积
            C[i+j+1]+=C[i+j]/10;//进位
            C[i+j]%=10;//取余
        }
    }
    int lc=3999;
    while(lc && C[lc]==0)lc--;//处理前导0
    for(int i=lc;i>=0;i--)
    {
        cout<<C[i];
    }
    return 0;
}
```

我来详细解释一下这个**位置关系**，比如计算123×45，123存成[3,2,1]，45存成[5,4]。3×5=15存到C[0]=5,C[1]=1；3×4=12加上之前的进位1变成13，存到C[1]=3,C[2]=1；2×5=10存到C[1]+=10变成C[1]=3+10=13，进位后C[2]=1+1=2，C[1]=3；2×4=8加上进位C[2]+=8变成C[2]=2+8=10，进位后C[3]=1，C[2]=0；1×5=5加上进位C[2]+=5变成C[2]=5；1×4=4加上进位C[3]+=4变成C[3]=5。最终C数组是[5,3,5,5]，倒序输出得到5535，验证一下123×45确实是5535。

关于**结果数组的大小为什么要开la+lb**呢，因为两个数相乘最大位数是两个数的位数之和，比如99×99=9801，三位数乘以三位数最多得到四位数，所以结果数组开la+lb就够了。

## 高精度除法（高精度/低精度）

**计算被除数->存商->求余数**

这里要特别强调一下，除法是**高精度除以低精度**，不是高精度除以高精度。为什么呢，因为高精度除以高精度的算法非常复杂，一般比赛里面也不会让你写，高精度除以低精度就简单多了。

代码如下

```cpp
#include <bits/stdc++.h>
using namespace std;
int A[5010];
int  C[5010];
int b;//高精度除以低精度
signed main()
{
    string a;
    cin>>a>>b;
    int la=a.size();
    for(int i=0;i<la;i++)
    {
        A[i]=a[la-i-1]-'0';
    }
    int r=0;//被除数
    for(int i=la-1;i>=0;i--)//从高位向低位
    {
        r=r*10+A[i];//计算被除数
        C[i]=r/b;//存商
        r%=b;//求余
    }
    int lc=la;
    while(lc && C[lc]==0)lc--;//处理前导0
    for(int i=lc;i>=0;i--)
    {
        cout<<C[i];
    }
    return 0;
}
```

**除法的逻辑和加减乘都不一样，是从高位向低位处理的**。r是余数，初始化为0。然后对于每一位，**把上一位的余数乘以10加上当前位的数字，作为新的被除数**，用这个被除数除以b就得到当前位的商，余数继续留到下一位。

举个例子，计算123÷7。从最高位开始，r=0，r=r*10+1=1，1/7=0余1，所以最高位商是0；然后r=1，r=r*10+2=12，12/7=1余5，所以第二位商是1；然后r=5，r=r*10+3=53，53/7=7余4，所以第三位商是7。最终商是017也就是17，余数是4，验证一下17×7+4=119+4=123，正确。

---

## 总结

到这里四种高精度算法就讲完了，简单总结一下：

**加法：低位到高位，累加进位取余**
**减法：低位到高位，借位求差，记得处理负数**
**乘法：低位到高位，注意i+j的位置关系**
**除法：高位到低位，用余数构建被除数**

四种算法的**数组大小**也很重要，**加法开max(la,lb)+1，减法开max(la,lb)，乘法开la+lb，除法开la**。

记住这些要点，高精度算法就不难了。
