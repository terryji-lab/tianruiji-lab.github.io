# 对顶堆
使用一个大根堆和一个小根堆来维护所有元素中第k大/小的元素。
以找第k大的元素为例：
如果输入的数据大于小根堆的根或者小根堆为空，就插入小根堆，否则加入大根堆。因为如果插入数据还没有小根堆的根大，那么一定比小根堆的所有元素都要小。**在输入过程中，如果小根堆的元素不足k个，就从大根堆转移最大的元素，也就是大根堆的根过去，如果超过k个元素，就把小根堆最小的元素，也就是小根堆的根转移到大根堆去，总之要维护小根堆在最后有k个元素，并且小根堆的根就是第k大的数**。
## 代码
```cpp
#include <bits/stdc++.h>
using namespace std;
priority_queue<int> bheap;//大顶堆
priority_queue<int,vector<int>,greater<int>> sheap;//小顶堆
int main()
{
    int n,k;
    cin>>n>>k;
    for(int i=1;i<=n;i++)
    {
        int x;
        cin>>x;
        if(sheap.empty() || x>sheap.top())sheap.push(x);//如果小顶堆为空或者大于小顶堆的根，就加入小顶堆
        else bheap.push(x);//否则加入大顶堆
        while(sheap.size()>k)bheap.push(sheap.top()),sheap.pop();//多于k个就转移到大顶堆
        while(sheap.size()<k && !bheap.empty())sheap.push(bheap.top()),bheap.pop();//少于k个就转移到小顶堆
    }
    cout<<sheap.top();
    return 0;
}
```
