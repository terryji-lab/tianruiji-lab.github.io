# 堆
堆是一种常见的数据结构，可以维护数据的最大或者最小值，在cpp中一般使用STL``priority_queue``来实现。这里我会讲解实现的代码。
## 存储结构
堆是一颗完全二叉树并且所有儿子节点都小于/大于父亲节点，所以根节点的数据就是整组数据的最小/最大值。因为是完全二叉树所以我们可以采用数组来保存一个堆，节点``u``的父亲节点就是``u/2``，儿子节点就是``2*u``和``2*u+1``。
## 插入操作
先插入到完全二叉树的末尾，然后进行上浮操作。
所谓上浮操作，就是判断当前节点和父亲节点的大小关系，以小根堆为例，如果儿子节点的值小于父亲节点的值，那么就和父亲节点位置交换，然后继续向上判断，直到找到根为止。
## 删除操作
我们用完全二叉树的最后一个节点来替代根节点，然后进行下潜操作。还是以小根堆为例，如果交换后新的根节点大于它的儿子节点就和儿子节点交换位置，直到到达叶子位置或者两个儿子节点都大于它为止。
## 代码
下面是小根堆的实现代码
```cpp
#include <bits/stdc++.h>
using namespace std;
int heap[100005];
int siz;
void up(int u)//上浮操作
{
    if(u/2 && heap[u/2]>heap[u])//u/2=0说明u的父亲节点是根节点
    {
        swap(heap[u],heap[u/2]);//交换位置
        up(u/2);//继续递归
    }
}
void heap_push(int x)
{
    heap[++siz]=x;//加入堆中
    up(siz);//上浮
}
void down(int u)//下潜操作
{
    int v=u;
    if(u*2<=siz && heap[u*2]<heap[v])v=u*2;
    if(u*2+1<=siz && heap[u*2+1]<heap[v])v=2*u+1;//v是小于u的儿子，这里左右儿子的判断顺序无所谓
    if(u!=v)//如果u==v说明没有小于u的儿子
    {
        swap(heap[u],heap[v]);//交换位置
        down(v);//继续下潜
    }
}
void heap_pop()
{
    heap[1]=heap[siz--];//完全二叉树最后位置的节点换到根节点
    down(1);//下潜
}
int n;
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        int x;
        cin>>x;
        heap_push(x);
    }
    for(int i=1;i<=n;i++)
    {
        cout<<heap[1]<<' ';
        heap_pop();
    }
    return 0;
}
```
