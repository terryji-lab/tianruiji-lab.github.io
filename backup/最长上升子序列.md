# 最长上升子序列
## 朴素做法
### DP数组含义 
``dp[]``数组的含义**是以当前下标为最后一个元素的最长上升子序列长度**，注意**子序列元素间是不连续的**。
### 初始化
**每个元素所构成的子序列长度至少为``1``**，所以整个``dp[]``数组都初始化为``1``。
### 递推公式
从左往右扫描，``i``代表当前元素的下标，``j``代表扫描元素的下标，``a[]``是存储数据的数组，如果``a[i]>a[j]``，那么更新``dp[]``数组，最后整个数组中的最大值就是答案。
- ``dp[i]=max(dp[j]+1,dp[i])``
### 代码
```cpp
#include <bits/stdc++.h>
using namespace std;
int dp[5010];
int a[5010];
int n,maxn;
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        dp[i]=1;
        cin>>a[i];
    }
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<i;j++)//j是i之前的下标
        {
            if(a[i]>a[j])
            {
                dp[i]=max(dp[i],dp[j]+1);//更新dp数组
            }
        }
        maxn=max(maxn,dp[i]);//保存最大值
    }
    cout<<maxn;
    return 0;
}
```
但是显然这种做法的时间复杂度太高了，所以接下来我会介绍一种二分优化的版本。

## 二分做法
### 介绍
我们维护一个``tail[]``数组来保存目前最长上升子序列的**长度**(注意：这个数组中的元素并不是最长上升子序列的元素，但是这个数组的长度就是最长上升子序列的长度)。我们扫描``a[]``数组，每遇到一个元素，如果比``tail[]``中的所有元素都要大，就加入``tail[]``数组，否则就替换掉``tail[]``中第一个大于等于它的元素。**替换掉过后这个位置的元素变得更小，更有机会组成更长的子序列，你可以把``tail[]``数组想成同时存储了多个子序列，``tail[]``大的长度就是其中最长的那一个，另外的都在``tail[]``内部进行更新**。因为``tail[]``数组是单调的，自然可以使用二分来做。

### 代码
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=5e4+5;
int a[N];
int tail[N];
int n,len;//len记录tail长度
int find(int v)
{
    int l=0,r=len+1;
    while(l+1<r)
    {
        int mid=l+(r-l)/2;
        if(tail[mid]>=v)r=mid;//查找第一个大于等于v的元素
        else l=mid;
    }
    return r;
}
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++)cin>>a[i];
    for(int i=1;i<=n;i++)
    {
        int pos=find(a[i]);
        if(pos==len+1)//如果大于所有元素
        {
            tail[++len]=a[i];//加入tail
        }
        else
        {
            tail[pos]=a[i];//否则替换
        }
    }
    cout<<len;//输出长度
    return 0;
}
```
