# 并查集
并查集是一种**树形**数据结构，支持两种操作：
- 合并：将两个子集合合并成一个集合
- 查找：确定某个元素处在哪个集合

一般使用数组来实现，``fa[]``是父亲节点数组。

## 初始化
刚开始没有任何合并操作，每个节点都是一个单独的集合，**所以每一个节点的父亲节点就是它自己**：
``for(int i=1;i<=n;i++)fa[i]=i;``

## 查找
根节点是集合的代表，查找就是找到元素所在集合的根。

1. 如果**父亲节点等于节点自己**，说明该节点就是根，直接返回。
2. 如果没有找到根，就**递归调用查找父亲节点的根**。
3. 找到后**直接修改当前节点的父亲节点到根上**，达到降低树的深度，路径压缩的效果。

```cpp
int find(int u)
{
	if(fa[u]==u)return u;
	return fa[u]=find(fa[u]);//路径压缩
}
```
## 合并
把一个集合的根指向另一个集合的根。
```cpp
void unionset(int u,int v)
{
	fa[find(u)]=find(v);
}
```
### 按秩合并
按照树的大小，**将小树合并到大树上，可以进一步减小树的深度，而减短路径**，这就是按秩合并。
```cpp
vector<int> sz(N,1);//每个集合的初始大小为1
void unionset(int u,int v)
{
	u=find(u),v=find(v);//找到u和v的根节点
	if(u==v)return;//如果在同一个集合内不需要合并
	if(sz[u]>sz[v])swap(u,v);//保证u是小树
	fa[u]=v;//把小树u的根指向大树v的根
	sz[v]+=sz[u];//大树的节点数增加
}
```
