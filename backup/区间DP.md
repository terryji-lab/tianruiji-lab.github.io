# 区间DP
## DP数组含义
区间dp中一般使用二维数组``dp[i][j]``来表示区间``i~j``的状态。
## 递推公式
区间dp一般分三步走：

1. 先枚举区间长度
2. 枚举区间起点
3. 枚举区间分割点

我们的思路就是一种分治思想，先把整个dp数组分成很多小区间，再慢慢将小区间进行合并最后得到整个区间的最优解。
## 例题
# P1775 石子合并（弱化版）

## 题目描述

设有 $N(N \le 300)$ 堆石子排成一排，其编号为 $1,2,3,\cdots,N$。每堆石子有一定的质量 $m_i\ (m_i \le 1000)$。现在要将这 $N$ 堆石子合并成为一堆。每次只能合并相邻的两堆，合并的代价为这两堆石子的质量之和，合并后与这两堆石子相邻的石子将和新堆相邻。合并时由于选择的顺序不同，合并的总代价也不相同。试找出一种合理的方法，使总的代价最小，并输出最小代价。

## 输入格式

第一行，一个整数 $N$。

第二行， $N$ 个整数 $m_i$。

## 输出格式

输出文件仅一个整数，也就是最小代价。

## 输入输出样例 #1

### 输入 #1

```
4
2 5 3 1
```

### 输出 #1

```
22
```
## 分析
我们使用二维数组``dp[i][j]``来表示合并``i``到``j``这个区间内所有石头所需要的最小代价。初始化的时候``dp[i][i]``全部初始化未``0``，因为和自己合并不需要付出代价，其余的全部初始化未正无穷方便后续状态更新。我们合并两堆石头的代价是两堆石头的重量之和，所以我们可以使用前缀和来快速计算石头的重量和。代码如下：
```cpp
#include <bits/stdc++.h>
#define int long long
using namespace std;
const int N=305;
int dp[N][N];
int pre[N];
int n;
signed main()
{
    cin>>n;
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=n;j++)
        {
            if(i==j)dp[i][j]=0;//自己和自己合并代价为0
            else dp[i][j]=1e9;//其余为正无穷
        }
    }
    for(int i=1;i<=n;i++)
    {
        cin>>pre[i];
        pre[i]+=pre[i-1];//计算前缀和数组
    }
    for(int len=2;len<=n;len++)//枚举区间长度，从2开始
    {
        for(int l=1;l+len-1<=n;l++)//枚举区间起点
        {
            int r=l+len-1;//计算区间重点
            for(int i=l;i<r;i++)//枚举区间分割点
            {
                dp[l][r]=min(dp[l][r],dp[l][i]+dp[i+1][r]+pre[r]-pre[l-1]);//状态转移，pre[r]-pre[l-1]是合并付出的代价，dp[l][i]+dp[i+1][r]是两个子区间之前的代价，合起来就是整个l~r区间的代价
            }
        }
    }
    cout<<dp[1][n];//输出合并整个区间的最小代价
    return 0;
}

```
