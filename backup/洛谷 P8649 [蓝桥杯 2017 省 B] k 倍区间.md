# P8649 [蓝桥杯 2017 省 B] k 倍区间

## 题目描述

给定一个长度为 $N$ 的数列， $A_1,A_2, \cdots A_N$，如果其中一段连续的子序列 $A_i,A_{i+1}, \cdots A_j(i \le j)$ 之和是 $K$ 的倍数，我们就称这个区间 $[i,j]$ 是 $K$ 倍区间。

你能求出数列中总共有多少个 $K$ 倍区间吗？

## 输入格式

第一行包含两个整数 $N$ 和 $K$ $(1 \le N,K \le 10^5)$。

以下 $N$ 行每行包含一个整数 $A_i$ $(1 \le A_i \le 10^5)$。

## 输出格式

输出一个整数，代表 $K$ 倍区间的数目。

## 输入输出样例 #1

### 输入 #1

```
5 2
1  
2  
3  
4  
5  
```

### 输出 #1

```
6
```

## 说明/提示

时限 2 秒, 256M。蓝桥杯 2017 年第八届

## 分析
求连续区间和自然想到会用**前缀和**。但是如果这道题使用两重循环暴力枚举区间端点的话会TLE，所以我们引入**哈希表**来解决这个问题。思路就是我们用哈希表记录在每一个``r``之前有多少个``l``，满足``[l,r]``区间的和是k的倍数，就可以化为线性复杂度。这里有一个细节，**要求是``(pre[r]-pre[l-1])%k==0``,可以转化为``pre[r]%k-pre[l-1]%k==0``**。代码如下
```cpp
#include <bits/stdc++.h>
using namespace std;
typedef long long ll;
ll n,k,ans;
ll pre[100010];
unordered_map<int,int> table;
int main()
{
    cin>>n>>k;
    for(int i=1;i<=n;i++)
    {
        cin>>pre[i];
        pre[i]=pre[i-1]+pre[i];//计算前缀和数组
    }
    for(int i=0;i<=n;i++)//从0开始遍历，相当于table[0]=1，然后从1开始遍历
    {
        ans+=table[pre[i]%k];//i前面有多少个满足条件的节点，就有多少个满足条件的区间
        table[pre[i]%k]++;//先计算后更新哈希表，防止重复计算自身
    }
    cout<<ans;
    return 0;
}
```
