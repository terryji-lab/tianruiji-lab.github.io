# 拓扑排序
给定一张**有向无环图**（DAG），排出所有顶点的一个序列A满足：
对于图中每条有向边（x，y），x在A中都出现在y之前，则称A是该图顶点的一个拓扑序。
拓扑排序可以**判断有向图中是否有环**，可以生成拓扑序列。
## Kahn算法
算法的核心是用队列维护一个入度为0的节点的集合。

1. 初始化，队列q压入所有入度为0的点
2. 每次从q中取出一个点x放入数组tp
3. 然后将x的所有出边删除。若将（x，y）删除后，y的入度变为0，则将y压入q中
4. 不断重复2，3步，直到q为空
5. 判断tp的元素个数是否等于n，是则有拓扑序，否则存在环

代码的实现如下：
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=100;
vector<int> e[N],tp;
int din[N];
int n,m;
bool toposort()
{
    queue<int>q;
    for(int i=1;i<=n;i++)
    {
        if(!din[i])q.push(i);
    }
    while(!q.empty())
    {
        auto x=q.front();
        q.pop();
        tp.push_back(x);
        for(auto&y:e[x])
        {
            if(--din[y]==0)q.push(y);
        }
    }
    return tp.size()==n;
}
int main()
{
    cin>>n>>m;
    for(int i=1;i<=m;i++)
    {
        int u,v;
        cin>>u>>v;
        e[u].push_back(v);
        din[v]++;
    }
    if(!toposort())cout<<-1;
    else for(auto&x:tp)cout<<x<<' ';
    return 0;
}
```
