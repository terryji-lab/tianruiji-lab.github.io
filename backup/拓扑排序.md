# 拓扑排序
给定一张**有向无环图**（DAG），排出所有顶点的一个序列A满足：
对于图中每条有向边（x，y），x在A中都出现在y之前，则称A是该图顶点的一个拓扑序。
拓扑排序可以**判断有向图中是否有环**，可以生成拓扑序列。
## Kahn算法
算法的核心是用队列维护一个入度为0的节点的集合。

1. 初始化，队列q压入所有入度为0的点
2. 每次从q中取出一个点x放入数组tp
3. 然后将x的所有出边删除。若将（x，y）删除后，y的入度变为0，则将y压入q中
4. 不断重复2，3步，直到q为空
5. 判断tp的元素个数是否等于n，是则有拓扑序，否则存在环

代码的实现如下：
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=100;
vector<int> e[N],tp;
int din[N];
int n,m;
bool toposort()
{
    queue<int>q;
    for(int i=1;i<=n;i++)
    {
        if(!din[i])q.push(i);
    }
    while(!q.empty())
    {
        auto x=q.front();
        q.pop();
        tp.push_back(x);
        for(auto&y:e[x])
        {
            if(--din[y]==0)q.push(y);
        }
    }
    return tp.size()==n;
}
int main()
{
    cin>>n>>m;
    for(int i=1;i<=m;i++)
    {
        int u,v;
        cin>>u>>v;
        e[u].push_back(v);
        din[v]++;
    }
    if(!toposort())cout<<-1;
    else for(auto&x:tp)cout<<x<<' ';
    return 0;
}
```

## 染色法
如果是Kahn算法是基于Bfs的算法，那么染色算法就是基于Dfs的算法。这种算法我们不用保存入度，替代的是一个``c[x]``数组来保存当前``x``的颜色。算法的核心是如果存在拓扑序，每个节点都会经历``0 --> -1--> 1``三种颜色。

1. 初始状态，所有点染色为0
2. 枚举每个点，进入x点，把x点染色为-1。然后枚举x的儿子y，如果y的颜色是0，就进入y继续走
3. 如果枚举完x的儿子，没发现环，就把x染色为1，并把x压入tp数组
4. 如果发现在递归过程中儿子节点的颜色为-1，则说明回到了祖先节点，发现了环，返回false退出程序。

- 0表示未被访问
- -1表示正在被访问
- 1表示访问完毕

这样做出来是从儿子节点push进tp的，所以要倒序输出才是正向拓扑序。
代码如下：
```cpp
#include <bits/stdc++.h>
using namespace std;
const int N=100;
vector<int> tp;
vector<int> e[N];
int c[N];
int n,m;
bool dfs(int x)
{
    c[x]=-1;//染色为-1
    for(auto&y:e[x])//枚举儿子节点
    {
        if(c[y]==-1)return false;//如果有正在访问中的点说明有环
        else if(!c[y])
        {
            if(!dfs(y))return false;//如果递归搜索子节点遇到环也返回false
        }
    }
    c[x]=1;//染色为1
    tp.push_back(x);//加入序列
    return true;
}
bool toposort()
{
    memset(c,0,sizeof(c));//初始染色为0
    for(int i=1;i<=n;i++)//枚举每一个节点
    {
        if(!c[i])//如果是0就进行访问
        {
            if(!dfs(i))return false;//存在环直接返回false
        }
    }
    reverse(tp.begin(),tp.end());//反转顺序
    return true;
}
int main()
{
    cin>>n>>m;
    for(int i=1;i<=m;i++)
    {
        int u,v;
        cin>>u>>v;
        e[u].push_back(v);
    }
    if(!toposort())cout<<-1;
    else for(auto&ele:tp)cout<<ele<<' ';
    return 0;
}
```
